       - Group files by lowercased stem. For each group:
         - If the group has a single file, insert `stem -> rel_path`.
         - If the group has multiple files, resolve from vault root `""` via `resolve_link(stem, "", all_rel_paths)` and insert the result (shortest-path winner; deterministic).
       - Keys are lowercased stems; values are vault-relative paths.
       - Used by `get_resolved_links` IPC command to populate the frontend click-handler map.
