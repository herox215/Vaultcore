   **Algorithm (line-level, per D-11 and RESEARCH Pattern 3):**

   ```rust
   pub fn three_way_merge(base: &str, left: &str, right: &str) -> MergeOutcome {
       // Shortcut: if left == right, no merge needed
       if left == right { return MergeOutcome::Clean(left.to_string()); }
       // If base == left, only external changed
       if base == left { return MergeOutcome::Clean(right.to_string()); }
       // If base == right, only local changed
       if base == right { return MergeOutcome::Clean(left.to_string()); }

       let base_lines: Vec<&str> = base.lines().collect();
       let left_lines: Vec<&str> = left.lines().collect();
       let right_lines: Vec<&str> = right.lines().collect();

       // Compute diffs from base to each branch
       let left_ops = capture_diff_slices(Algorithm::Myers, &base_lines, &left_lines);
       let right_ops = capture_diff_slices(Algorithm::Myers, &base_lines, &right_lines);

       // Extract changed base-line ranges from each branch
       let left_ranges = changed_base_ranges(&left_ops, &base_lines);
       let right_ranges = changed_base_ranges(&right_ops, &base_lines);

       // Check for overlap
       let has_conflict = left_ranges.iter().any(|lr| {
           right_ranges.iter().any(|rr| ranges_overlap(lr, rr))
       });

       if has_conflict {
           // D-11 step 4: keep local (left) version entirely
           MergeOutcome::Conflict(left.to_string())
       } else {
           // Apply right's changes onto left
           let merged = apply_non_conflicting(
               &base_lines, &left_lines, &right_lines,
               &left_ops, &right_ops
           );
           MergeOutcome::Clean(merged)
       }
   }
   ```

   **Helper functions:**

   `changed_base_ranges(ops, base_lines) -> Vec<(usize, usize)>`:
   - Walk through diff operations, track base line index.
   - For each `ChangeTag::Delete` or `ChangeTag::Equal` entry, advance base index.
   - Record `(start, end)` ranges of base lines that were deleted or replaced.

   `ranges_overlap(a: &(usize, usize), b: &(usize, usize)) -> bool`:
   - `a.0 < b.1 && b.0 < a.1` (standard interval overlap check).

   `apply_non_conflicting(base, left, right, left_ops, right_ops) -> String`:
   - Build the merged output by walking base lines.
   - For each base line: if right changed it (and left didn't), use right's version. If left changed it (and right didn't), use left's version. If neither changed it, use base.
   - Handle insertions (lines added by right that don't correspond to base lines) by inserting them at the correct position.
   - Join with `"\n"` and handle trailing newline preservation.
