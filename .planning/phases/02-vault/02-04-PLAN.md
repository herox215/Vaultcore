   **spawn_watcher function:**
   ```rust
   pub fn spawn_watcher(
       app: AppHandle,
       vault_path: PathBuf,
       write_ignore: Arc<Mutex<WriteIgnoreList>>,
   ) -> Debouncer<RecommendedWatcher, notify_debouncer_full::FileIdMap>
   ```
   - Create debouncer with `DEBOUNCE_DURATION` (200ms per D-10).
   - `debouncer.watch(&vault_path, RecursiveMode::Recursive)`.
   - In the callback closure:
     a. On `Ok(events)`: call `process_events(&app, &write_ignore, &vault_path, events)`.
     b. On `Err(errors)`: for each error, emit `vault://watcher_error`. If error indicates path not found on vault root, emit `vault://vault_status { reachable: false }`.

   **process_events function:**
   ```rust
   fn process_events(
       app: &AppHandle,
       write_ignore: &Arc<Mutex<WriteIgnoreList>>,
       vault_path: &Path,
       events: Vec<DebouncedEvent>,
   )
   ```
   Steps:
   1. Filter events: skip any whose path starts with a dot-component (e.g., `.obsidian/`, `.trash/`, `.git/`). Check each component of the relative path from vault_path.
   2. Filter self-writes: lock write_ignore, call `should_ignore(&event.path)` for each event. Skip if true.
   3. After filtering, count remaining events. If count > `BULK_THRESHOLD`:
      - Emit `vault://bulk_change_start { estimated_count: count }`.
      - Process all events (emit individual `vault://file_changed` for each).
      - Use `rayon::par_iter()` if batch processing is needed (SYNC-04). For Phase 2, the "batch processing" is just emitting events; rayon parallelism applies to future indexing. For now, emit events sequentially.
      - Emit `vault://bulk_change_end {}`.
   4. If count <= threshold: emit individual `vault://file_changed` events.

   **Map event kinds:**
   - `EventKind::Create(_)` -> kind "create"
   - `EventKind::Modify(ModifyKind::Data(_))` -> kind "modify"
   - `EventKind::Modify(ModifyKind::Name(RenameMode::Both))` -> kind "rename" (with new_path from event.paths[1])
   - `EventKind::Remove(_)` -> kind "delete"
   - Other modify kinds: map to "modify" as safe default.
