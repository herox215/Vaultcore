   - Return `Result<T, VaultError>`

   **create_file(state, parent: String, name: String) -> String:**
   - If `name` is empty, default to `"Untitled.md"`.
   - If file exists, auto-suffix: `"Untitled 1.md"`, `"Untitled 2.md"`, etc.
   - Ensure parent is inside vault.
   - `std::fs::write(final_path, "")` to create empty file.
   - Record `final_path` in write_ignore.
   - Return the absolute path of the created file.

   **create_folder(state, parent: String, name: String) -> String:**
   - If `name` is empty, default to `"New Folder"`.
   - Auto-suffix on collision: `"New Folder 1"`, `"New Folder 2"`, etc.
   - `std::fs::create_dir(final_path)`.
   - Record in write_ignore.
   - Return absolute path.

   **rename_file(state, old_path: String, new_name: String) -> RenameResult:**
   ```rust
   #[derive(Serialize)]
   pub struct RenameResult {
       pub new_path: String,
       pub link_count: u32,
   }
   ```
   - Compute new_path = old_path.parent().join(new_name).
   - Validate both old and new paths inside vault.
   - Count wiki-links: scan all `.md` files in vault for regex `\[\[old_filename_without_ext\]\]` (case-sensitive). Use `walkdir` filtered by `is_excluded`. Count occurrences across all files.
   - `std::fs::rename(old_path, new_path)`.
   - Record both old and new paths in write_ignore.
   - Return RenameResult { new_path, link_count }.

   **delete_file(state, path: String) -> ():**
   - Validate path inside vault.
   - Compute trash_dir = vault_root / ".trash/". `create_dir_all` if not exists.
   - Compute dest = trash_dir / filename. If exists, auto-suffix.
   - `std::fs::rename(path, dest)` (move, not delete).
   - Record both source and dest paths in write_ignore (Pitfall 6 from RESEARCH).
   - Return Ok(()).

   **move_file(state, from: String, to_folder: String) -> String:**
   - Validate both `from` and `to_folder` inside vault.
   - Compute dest = to_folder / from.file_name().
   - If dest exists, return error (FileExists — use VaultError::Io with appropriate message).
   - `std::fs::rename(from, dest)`.
   - Record both paths in write_ignore.
   - Return dest absolute path.

   **count_wiki_links(state, filename: String) -> u32:**
   - Get vault root from state.
   - Walk all .md files (reuse vault.rs walk pattern with is_excluded filter).
   - For each file, read contents and count occurrences of `[[filename_stem]]` where filename_stem is the filename without .md extension.
   - Return total count.
