**1. Create `src-tauri/src/commands/search.rs`** with three Tauri commands:

**search_fulltext command:**
```rust
#[derive(Serialize, Clone, Debug)]
#[serde(rename_all = "camelCase")]
pub struct SearchResult {
    pub path: String,        // absolute path
    pub title: String,
    pub score: f32,
    pub snippet: String,     // HTML with <b>highlighted</b> terms
    pub match_count: usize,  // number of term matches in this doc
}

#[tauri::command]
pub async fn search_fulltext(
    query: String,
    limit: usize,
    state: tauri::State<'_, VaultState>,
) -> Result<Vec<SearchResult>, VaultError>
```

Implementation:
- Get `IndexCoordinator` from `state.index_coordinator`. If `None`, return empty vec (no vault open).
- Access `coordinator.reader()` and `coordinator.index()`.
- Create `QueryParser::for_index(&index, vec![title_field, body_field])`.
- Parse with `query_parser.parse_query_lenient(&query)` — use the `.0` (parsed query), log `.1` (errors) at debug level. Never return error to user for bad syntax (D-03 live typing).
- Search with `searcher.search(&query, &TopDocs::with_limit(limit))`.
- Create `SnippetGenerator::create(&searcher, &*query, body_field)`. Call `snippet_gen.set_max_num_chars(200)`.
- For each `(score, doc_address)`: retrieve document, extract `path_field` stored value, `title_field` stored value, generate snippet HTML via `snippet.to_html()`.
- Cap results at `limit` (default 100 per D-08). Return `Vec<SearchResult>`.
- Wrap `Index::open` / `Searcher::search` errors: catch `TantivyError`, map to `VaultError::IndexCorrupt`.

**search_filename command:**
```rust
#[derive(Serialize, Clone, Debug)]
#[serde(rename_all = "camelCase")]
pub struct FileMatch {
    pub path: String,         // relative path
    pub score: u32,           // nucleo score
    pub match_indices: Vec<u32>,  // character indices that matched (for highlight)
}

#[tauri::command]
pub async fn search_filename(
    query: String,
    limit: usize,
    state: tauri::State<'_, VaultState>,
) -> Result<Vec<FileMatch>, VaultError>
```

Implementation:
- Get `FileIndex` from coordinator, call `all_relative_paths()`.
- Get pre-warmed `Matcher` from coordinator (Pitfall 6).
- Create `Pattern::parse(&query, CaseMatching::Ignore, Normalization::Smart)`.
- For each path, call `pattern.score(Utf32Str::new(&path, &mut buf), &mut matcher)`. Collect `(score, index)` pairs where `score.is_some()`.
- Also call `pattern.indices(...)` to get `match_indices` for frontend highlight.
- Sort by score descending. Take first `limit` results.
- Return `Vec<FileMatch>`.

**rebuild_index command:**

#[tauri::command]
pub async fn rebuild_index(

Implementation:

- After rebuild completes (await a oneshot channel response from the queue consumer), emit `{ message: "Index aktualisiert", variant: "clean-merge" }`.

**3. Create `src/store/searchStore.ts`** — classic Svelte `writable` store per D-06/RC-01:

```typescript
import { writable } from "svelte/store";
import type { SearchResult } from "../types/search";

export interface SearchStoreState {
  query: string;
  results: SearchResult[];
  totalMatches: number;       // total before 100-cap (from result array length or Tantivy count)
  totalFiles: number;         // unique files matched
  isSearching: boolean;
  isRebuilding: boolean;
  activeTab: "files" | "search";  // sidebar tab state (D-01)
}

const initial: SearchStoreState = {
  query: "",
  results: [],
  totalMatches: 0,
  totalFiles: 0,
  isSearching: false,
  isRebuilding: false,
  activeTab: "files",
};

function createSearchStore() {
  const { subscribe, set, update } = writable<SearchStoreState>(initial);

  return {
    subscribe,
    setQuery: (query: string) => update((s) => ({ ...s, query })),
    setResults: (results: SearchResult[], totalFiles: number) =>
      update((s) => ({
        ...s,
        results,
        totalMatches: results.length,
        totalFiles,
        isSearching: false,
      })),
    setSearching: (isSearching: boolean) => update((s) => ({ ...s, isSearching })),
    setRebuilding: (isRebuilding: boolean) => update((s) => ({ ...s, isRebuilding })),
    setActiveTab: (tab: "files" | "search") => update((s) => ({ ...s, activeTab: tab })),
    clearResults: () =>
      update((s) => ({ ...s, results: [], totalMatches: 0, totalFiles: 0, query: "" })),
    reset: () => set(initial),
  };
}

export const searchStore = createSearchStore();
```
